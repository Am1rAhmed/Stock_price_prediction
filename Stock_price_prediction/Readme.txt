This is a machine learning project for predcting stock prices of a company between 2018 to 2022.

While converting the data set from txt file to excel/csv please use only tab as delimiter.

To convert txt to excel:
1-Go to excel.
2- Go to open tab.
3-Then go to browse.
4-Select the txt file you need to convert.

Note if the txt file is not showing please select all files while going through step 3.

It has 7 coloumns:
1-Date.
2-Open.
3-High.
4-Low.
5-Close.
6-Adj Close.
7-Volume.
Total rows are 1009
